# Cloud Image Processing Pipeline

This project implements a serverless image processing pipeline using AWS services.

## 🚀 Features
- Automatic image processing on upload
- Serverless architecture
- Real-time email notifications
- Scalable and efficient system

## 🛠️ Tech Stack
- AWS S3
- AWS Lambda (Java)
- AWS SNS
- Maven

## ⚙️ Workflow
1. Upload image to S3 bucket
2. Lambda function is triggered
3. Image is processed (resized/transformed)
4. Output stored in another S3 bucket
5. SNS sends notification email
